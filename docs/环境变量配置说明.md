# 配置说明文档

## 环境变量配置

本项目使用环境变量管理敏感信息，避免将密钥提交到版本控制系统。

### 快速开始

1. 复制环境变量模板：
   ```bash
   cp .env.example .env
   ```

2. 编辑 `.env` 文件，填入真实的配置值：
   ```bash
   # 通义千问 API Key
   DASHSCOPE_API_KEY=sk-your-actual-api-key

   # 数据库配置
   DB_PASSWORD=your_actual_password

   # JWT 密钥
   JWT_SECRET=your_actual_jwt_secret
   ```

3. Spring Boot 会自动加载 `.env` 文件中的环境变量（如果使用 spring-boot-dotenv 依赖）

### 环境变量说明

| 变量名 | 说明 | 默认值 | 是否必填 |
|--------|------|--------|---------|
| `DASHSCOPE_API_KEY` | 通义千问 API 密钥 | 无 | ✅ 必填 |
| `DB_HOST` | MySQL 主机地址 | localhost | 可选 |
| `DB_PORT` | MySQL 端口 | 3306 | 可选 |
| `DB_NAME` | 数据库名称 | consultant | 可选 |
| `DB_USERNAME` | 数据库用户名 | root | 可选 |
| `DB_PASSWORD` | 数据库密码 | root123 | 可选 |
| `REDIS_HOST` | Redis 主机地址 | localhost | 可选 |
| `REDIS_PORT` | Redis 端口 | 6379 | 可选 |
| `JWT_SECRET` | JWT 签名密钥 | 默认密钥 | 生产环境必填 |

### 获取通义千问 API Key

1. 访问阿里云 DashScope 控制台：https://dashscope.console.aliyun.com/apiKey
2. 登录阿里云账号
3. 创建或查看你的 API Key
4. 复制 API Key 并填入 `.env` 文件

### 安全提醒

⚠️ **重要**：
- `.env` 文件已添加到 `.gitignore`，不会被提交到 Git
- 生产环境务必使用强随机字符串作为 JWT 密钥
- 不要在代码、文档、issue 中暴露真实的 API Key
- 定期更换密钥以提高安全性

### IDE 配置（可选）

如果使用 IDEA，可以在 Run Configuration 中配置环境变量：
1. Run → Edit Configurations
2. Environment variables：添加上述环境变量
3. 点击 OK

### Docker 部署

使用 Docker Compose 时，在 `docker-compose.yml` 的 `environment` 部分配置环境变量，或使用 `env_file` 指定 `.env` 文件。
