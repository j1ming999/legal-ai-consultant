# 项目经历

## 法律智能问答系统

**项目简介**：基于 RAG 技术的法律智能问答系统，支持法律法规的智能检索与问答，提供法条溯源功能。

**技术栈**：Spring Boot 3、LangChain4J、通义千问大模型、Redis（向量数据库+对话记忆）、MySQL、Vue 3

**核心工作**：
- 设计并实现混合检索策略，融合向量语义检索与关键词检索（RRF 算法），通过消融实验验证向量检索召回率达 87.2%，精确率达 91.1%
- 实现基于 LangChain4J 的 RAG 流程，包含查询改写、动态检索、流式响应，优化检索性能
- 设计用户系统与对话持久化方案，基于 JWT 实现鉴权，支持游客/登录双模式
- 实现法条溯源功能，展示回答所引用的法律条文出处，增强系统可信度
- 完成前后端开发与 Docker 容器化部署，系统响应时间控制在 200ms 以内

**项目成果**：系统已完成开发并实际部署，检索准确率和响应速度达到预期目标，支持生产环境使用
